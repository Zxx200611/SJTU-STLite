### 各个操作复杂度分析（二项堆）

+ push：均摊 $O(1)$，最坏 $O(\log)$。

  在没有 pop 的情况下，设 $2^{l-1}-1\le s \le 2^l-1$，则总时间复杂度不大于 push $2^l-1$ 个数的复杂度。

  在 push $2^l$ 个数的过程中，每次操作 $s \leftarrow s+1$，时间复杂度为 $O(\operatorname{popcount}(s\operatorname{xor} (s+1)))$，故单次操作复杂度为 $O(\log s)$。

  总时间复杂度为 $O(\sum_{s\le 2^l-1}\operatorname{popcount}(s\operatorname{xor} (s+1)))=O(1+2+4+\dots+2^l)=O(s)$。

+ pop：同理可知，在没有 push 的情况下也是均摊 $O(1)$。最坏 $O(\log)$。

+ top：$O(1)$。

+ merge：相当于用竖式计算两个二进制数相加，复杂度 $O(\log)$。
